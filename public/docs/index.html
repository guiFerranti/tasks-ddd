Introduction

API para gestão de tarefas colaborativas

Base URL: http://localhost:8000 
This documentation aims to provide all the information you need to work with our API.

<aside>As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile).
You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).</aside>

Authenticating requests

This API is not authenticated.

Autenticação

Endpoints para gerenciar autenticação de usuários (login/logout).

Login de Usuário

Autentica um usuário e retorna um token JWT.

Example request:
curl --request POST \
    "http://localhost:8000/api/auth/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"usuario@exemplo.com\",
    \"password\": \"senha123\"
}"

const url = new URL(
    "http://localhost:8000/api/auth/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "usuario@exemplo.com",
    "password": "senha123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "refresh_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
}

Example response (401):


{
    "error": "Credenciais inválidas"
}

Example response (422):


{
    "errors": {
        "email": [
            "O campo email é obrigatório"
        ],
        "password": [
            "O campo password é obrigatório"
        ]
    }
}
Received response:

Tarefas

Endpoints para gerenciamento de tarefas (CRUD, listagem com filtros).

Listar Tarefas com Filtros

requires authentication

Lista tarefas com filtragem avançada.

Example request:
curl --request GET \
    --get "http://localhost:8000/api/tasks?assignedTo=2&status=pending&createdAfter=2024-05-01" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8000/api/tasks"
);

const params = {
    "assignedTo": "2",
    "status": "pending",
    "createdAfter": "2024-05-01",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):


[

        "id": 1,
